Una aplicación de machine learning que clasifica automáticamente si un email es SPAM o no usando técnicas de procesamiento de lenguaje natural. Aprendi a como usar las ventanas en python

Usamos 
Python como lenguaje principal
scikit-learn para machine learning
Tkinter para la interfaz gráfica
Pandas/NumPy para manejo de datos
